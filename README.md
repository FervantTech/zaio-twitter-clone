# X (Twitter) Clone

A responsive Twitter/X homepage clone built as part of the Zaio Solo Twitter Project.

## Features

### Core Timeline Features

* Twitter-style left navigation bar
* Responsive timeline feed
* Tweet composer section
* Trending sidebar
* Suggested accounts to follow
* Sticky transparent header
* Responsive layout for desktop, tablet, and mobile devices

### Manual Feature (Built Without AI Assistance)

* Interactive like system

  * Toggle likes on posts
  * Dynamic like count updates
  * Visual liked state using a filled heart icon

### Cursor AI Features

* Feature 1: Explore page which opens when "Explore" button is clicked.
* Feature 2: Working modal with post that adds to timeline.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Cursor AI
* Netlify

## Responsive Design

The application has been optimized for:
